Farthest Point Sampling (FPS) in Point Clouds

Published: November 2025 Category: Point Cloud

Introduction

In 3D point cloud processing, we often want to downsample points while preserving coverage of the original shape. Farthest Point Sampling (FPS) is a popular algorithm for this. It selects points iteratively such that each new point is the farthest from the already chosen set.

FPS Algorithm

Given a point cloud ğ’« with N_in points and a target of N_out sampled points:

  1. Pick a random point from the cloud and add it to the sampled set ğ’«_s.
  2. For each point not in ğ’«_s, compute its distance to every point in ğ’«_s and take the minimum distance.
  3. Choose the point with the maximum minimum distance to add to ğ’«_s.
  4. Repeat until ğ’«_s has N_out points.

This ensures the selected points are well spread out across the point cloud.

Example

Let's take a simple 2D point cloud with 10 points:
