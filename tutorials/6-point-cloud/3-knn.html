k-Nearest Neighbors (k-NN) Grouping in Point Clouds

Published: November 2025 Category: Point Cloud

Introduction

After we perform Farthest Point Sampling (FPS) to select a subset of representative points from a point cloud, the next crucial step is to gather local neighborhoods around those points. This is achieved through the k-Nearest Neighbors (k-NN) algorithm.

In deep learning architectures like PointNet++ and DGCNN, this step allows each sampled point (called a center) to aggregate local geometric information from its nearest neighbors.

Grouping (k-Nearest Neighbors - k-NN)

For each sampled point (center), we find the 3 nearest points in the original cloud based on Euclidean distance.

This process creates local groups of points for each sampled center. Each group is then passed through shared MLPs (as in PointNet++) or graph edge convolutions (as in DGCNN) to extract local features.
